# 🚀 Real-Time Fraud Detection System

> Production-grade Machine Learning system for detecting fraudulent financial transactions in real-time.

---

## 📌 Overview

This project simulates a real-world fraud detection system used in fintech platforms.  
It processes transactions in real time, predicts fraud probability using ML, and logs results for monitoring and analysis.

---

## 🔥 Key Features

- ⚡ Real-time fraud prediction API (FastAPI)
- 🧠 Machine Learning model with probability scoring
- 📊 Interactive dashboard (Streamlit)
- 🗃️ Persistent logging with database (SQLAlchemy)
- 📈 Fraud analytics and monitoring
- 🔍 Explainable AI (SHAP - optional)
- 🐳 Dockerized deployment (production-ready)
