# 💡 Problem Statement

Financial fraud results in billions of losses annually.
Traditional systems fail due to **delayed batch processing**.

👉 This system enables:

* ⚡ Real-time fraud detection
* 🧠 ML-based decision making
* 📊 Live monitoring dashboard
